# GitHub Analytics Dashboard & API

An interactive dashboard and comprehensive API for in-depth analysis of GitHub user statistics.

## Overview

The GitHub Analytics Dashboard provides a visual and data-driven look at any GitHub user's profile. It aggregates data from multiple GitHub API endpoints to present a unified view of:

- **Profile Statistics**: Total commits, contribution streaks, and total stars.
- **Language Analysis**: Break down of programming languages used across repositories.
- **Contribution History**: Interactive graphs showing activity over time.
- **Repository Metadata**: Details on pinned, top-starred, and all public repositories.
- **Engagement**: Pull requests, organization contributions, and recent commit history.
